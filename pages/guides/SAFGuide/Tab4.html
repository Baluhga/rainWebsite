Bounty Sets

This section contains a list of sets to be used for various bounties that have equipment restrictions. These sets are NOT NORMAL PROGRESSION SETS  and many of them require a lot of muso decorations to make as well as a lot of farming for the armor pieces.

G-Rank Sets

CS+SW No Cuffs

This set works for all of Bronze Prog and any bounty that restricts you to G rank gear, no tech and no solid deter. Credits to Jek for cooking this up.

Weapon: Torag Canso+ (Diva)/Burning Zero Dragon Axe (Elz)/Eternal Creation+ (Wht.Fatalis)

Sigils:  SAF Up/(SAF) Guard/Atk + All Atk + All Atk

Guild Food:  All Res +20/Incitement

Poogie Skill:  Defence

SR Skills:  All Res/Ele Res/Affinity

Caravan Skills:  Weapon Art (lg) / Riser (lg)

Diva Skill: Vampirism+2

Fruit Skill: Shiri or Ele Resist

Reflect No Cuffs

Weapon: Torag Canso+ (Diva)/Burning Zero Dragon Axe (Elz)/Eternal Creation+ (Wht.Fatalis)

Sigils:  SAF Up/(SAF) Guard/Atk + All Atk + All Atk

Guild Food:  All Res +20/Incitement

Poogie Skill:  Taijutsu/Defence

SR Skills:  All Res/Ele Res/Affinity

Caravan Skills:  Weapon Art (lg) / Perfect Defence (lg)
